Esta es una prueba para GitHub.

Para adjuntar tu cuenta a la terminal debes poner: git config --global user.email "tu@ejemplo.com"

Para vincular tu repositorio local con tu repositorio en GitHub deben copiar el comando que tiene un link, aparece al crear tu repositorio en GitHub.

Los siguientes comandos se agregan en una terminal de VSC.

git init es para crear un repositorio.

git init -q es para crear un repositorio sin el mensaje de que se creo un nuevo repositorio.

git init -b unNombre es para crear un repositorio escogiendo el nombre EJ: git init -b test.

git init -b unNombre -q es para crear un repositorio escogiendo el nombre y sin el mensajede que se creo.

git status es para ver el estado de repositorio.

git add . es para guardar los archivos en el repositorio.

git comit -m "(descripci√≥n de la version o sus cambios a la anterior)" es para poder darle una descripcion a nustros cambios respecto a la version anterior.

git push es para guardar tu repositorio en la nube (al hacer tu primera git push te saldra un mensaje con un comando. Debes copiar ese comando y ponerlo en la terminal).
